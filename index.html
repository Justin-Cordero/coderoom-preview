Code Room

About

What is Code Room?

Code Room is a project aimed at improving the learning experience for new developers. By allowing educators to provide immediate access to online IDE's with preloaded coding challenges or lessons that can be worked in by groups in real-time in a remote setting with no overhead.

Why was Code Room created?

Code Room was created to solve a problem that we (the creators) experienced during our coding bootcamp. Our instructor wanted to demonstrate a coding concept that they were teaching by giving us a quick challenge to solve in small groups. However, we quickly realized that there wasn't an easy way for us to collaborate in real time to solve the problem, run our code, and communicate effectively without using multiple tools that required account creation or downloads. Code Room is our solution to that problem and it's still being used by instructors from our coding bootcamp for this very reason.
